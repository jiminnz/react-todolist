# react-todolist
> __React로 제작한 Todo List__
> 
> 입력창에 텍스트를 입력한 후 + 버튼을 클릭하면 리스트에 추가됩니다.  
> 입력하지않고 + 버튼을 클릭하면 알림창이 나옵니다.


- 사용 환경 : React
- 사용 기술    
  
    🌟 state
      
    🌟 map()  
      
    🌟 ref  
      
    🌟 if else()
